`Redis`哈希是一种数据类型，表示字符串字段和字符串值之间的映射关系。`Redis`哈希结构存储一组字段值对，设计上不占用太多空间，因此非常适合表示数据对象。它提供了添加、获取或删除单个项、获取整个哈希值或将哈希值中的一个或多个字段用作计数器的功能。
