`Redis`集合数据结构可存储一组唯一的成员。利用 Sets，你可以添加、获取或删除成员，检查成员资格，或检索随机成员。利用排序算法，你还可以执行集合操作，如交集、联合和集合差，并计算集合的万有引力。
