`Redis` HyperLogLog(**布隆过滤器**)是一种概率数据结构，用于在内存大小不变的情况下计算唯一值（设置卡入度）。你可以高效地添加和统计大量唯一项，并将两个或多个`HyperLogLog`数据结构合并为一个。
